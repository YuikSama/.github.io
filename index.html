  • 优化的界面布局，操作更加直观

🎮 使用指南

基本操作：

  1. 先运行游戏，再启动修改器
  2. 修改器会自动检测并连接游戏
  3. 设置想要的阳光/金币数值
  4. 勾选"无限"选项启用功能
  5. 勾选"自动锁定"保持数值恒定

热键自定义：

  1. 进入"设置"选项卡
  2. 点击热键输入框，按下想要的组合键
  3. 点击"应用热键设置"保存
  4. 使用自定义热键快速操作

默认热键参考：

  • 修改阳光: Ctrl + S
  • 切换无限阳光: Ctrl + Shift + S
  • 修改金币: Ctrl + C
  • 切换无限金币: Ctrl + Shift + C
  • 刷新游戏: F5
